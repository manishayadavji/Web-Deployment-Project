# 🍔 Food Delivery App (DevOps Project)

This is a simple food delivery website.

## 🚀 Features
- Food items UI (Pizza, Burger, etc.)
- Live running on AWS EC2
- Auto deploy using GitHub Actions (CI/CD)

## 🛠 Tech Stack
- HTML, CSS
- AWS EC2
- GitHub
- GitHub Actions

## ⚙️ How it works
1. I push code to GitHub  
2. GitHub Actions run automatically  
3. Connects to EC2 via SSH 
4. Website updates automatically
